# Younes Sefsaf - Personal Portfolio

A sleek, modern, dark-themed personal portfolio built with a vCard-style aesthetic. Designed to showcase software engineering projects, technical skills, and professional experience in an elegant, interactive format.

![Next.js](https://img.shields.io/badge/Next.js-15-black?style=for-the-badge&logo=next.js)
![React](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4-38B2AC?style=for-the-badge&logo=tailwind-css)

## 🚀 Features

- **Dark vCard Aesthetic**: A beautiful glassmorphic dark theme with striking gold accents.
- **Responsive Layout**: A sticky sidebar on desktop that elegantly stacks on mobile devices, paired with a dynamic bottom navigation bar for touch screens.
- **Interactive Timeline**: A custom-built timeline component for the Resume section to showcase education and project history.
- **Projects Grid**: A dedicated projects showcase with hover effects and responsive image placeholders.
- **Fast & Modern**: Built on the latest Next.js App Router and React 19 for optimal performance.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **UI Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & Custom inline SVGs

## getting Started

First, clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com/Younes-sef/porfolio.git

# Navigate to the project directory
cd portfolio

# Install dependencies
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `app/` - Next.js App Router layout, pages, and global CSS.
- `components/` - Reusable UI components:
  - `Sidebar.tsx` - The left-hand vCard profile panel.
  - `MainContent.tsx` - The main right-hand container with tab routing.
  - `About.tsx` - The professional summary section.
  - `Resume.tsx` - The interactive education and experience timelines.
  - `Projects.tsx` - The responsive grid showcasing recent work.
